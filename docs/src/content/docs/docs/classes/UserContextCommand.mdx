## Methods

### `setData(data: AnyContextCommandData)`

Sets the data for the slash command.

- Parameters:
  - `data`: The data to set for the slash command.
- Returns: The modified `UserContextCommand`.

### `setDevOnly(devOnly: boolean)`

Sets whether the command is for development purposes only.

- Parameters:
  - `devOnly`: A boolean indicating if the command is for development only.
- Returns: The modified `UserContextCommand`.

### `setGuildOnly(guildOnly: boolean)`

Sets whether the command can only be used in guilds (servers).

- Parameters:
  - `guildOnly`: A boolean indicating if the command can only be used in guilds.
- Returns: The modified `UserContextCommand`.

### `setUserPermissions(userPermissions: PermissionResolvable)`

Sets the required user permissions to use the command.

- Parameters:
  - `userPermissions`: The required user permissions.
- Returns: The modified `UserContextCommand`.

### `setBotPermissions(botPermissions: PermissionResolvable)`

Sets the required bot permissions to execute the command.

- Parameters:
  - `botPermissions`: The required bot permissions.
- Returns: The modified `UserContextCommand`.

### `setDeleted(deleted: boolean)`

Sets whether the command is deleted.

- Parameters:
  - `deleted`: A boolean indicating if the command is deleted.
- Returns: The modified `UserContextCommand`.

### `setRun(runRunFunction)`

Sets the function to be executed when the command is used.

- Parameters:
  - `run`: The function to execute when the command is invoked.
- Returns: The modified `UserContextCommand`.
