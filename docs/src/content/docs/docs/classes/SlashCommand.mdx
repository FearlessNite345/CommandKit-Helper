## Properties

#### `data`

- Type: [`AnySlashCommandData`](/commandkit-helper/docs/typedef/anyslashcommanddata)

- Type: [`CommandOptions`](https://commandkit.js.org/typedef/CommandOptions)

#### `run`

- Type: [`SlashCommandRunFunction`](/commandkit-helper/docs/typedef/slashcommandrunfunction)

## Methods

#### `setData(data)`

Sets the data for the slash command.

- Parameters:
  - `data`: [`AnySlashCommandData`](/commandkit-helper/docs/typedef/anyslashcommanddata)
- Returns: The modified `SlashCommand`.

#### `setDevOnly(devOnly)`

Sets whether the command is for development purposes only.

- Parameters:
  - `devOnly`: `boolean`
- Returns: The modified `SlashCommand`.

#### `setGuildOnly(guildOnly)`

Sets whether the command can only be used in guilds (servers).

- Parameters:
  - `guildOnly`: `boolean`
- Returns: The modified `SlashCommand`.

:::danger
  This is deprecated. Use `dm_permission` in your command `setData` instead.

  ```js
  export = new MessageContextCommand()
    .setData(
      new ContextMenuCommandBuilder()
        .setDMPermission(false)
    )
  ```
:::

#### `setUserPermissions(userPermissions)`

Sets the required user permissions to use the command.

- Parameters:
  - `userPermissions`: [`PermissionResolvable`](https://old.discordjs.dev/#/docs/discord.js/main/typedef/PermissionResolvable)
- Returns: The modified `SlashCommand`.

#### `setBotPermissions(botPermissions)`

Sets the required bot permissions to execute the command.

- Parameters:
  - `botPermissions`: [`PermissionResolvable`](https://old.discordjs.dev/#/docs/discord.js/main/typedef/PermissionResolvable)
- Returns: The modified `SlashCommand`.

#### `setDeleted(deleted)`

Sets whether the command is deleted.

- Parameters:
  - `deleted`: `boolean`
- Returns: The modified `SlashCommand`.

#### `addCustomOption(optionName, optionValue)`

Adds a custom option to the options section to be used in a custom validation
`commandObj.options?.optionname`

- Parameters:
  - `optionName`: `string`
  - `optionValue`: `any`
- Returns: The modified `SlashCommand`.

#### `setRun(run)`

Sets the function to be executed when the command is used.

- Parameters:
  - `run`: [`SlashCommandRunFunction`](/commandkit-helper/docs/typedef/slashcommandrunfunction)
- Returns: The modified `SlashCommand`.
